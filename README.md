# AlienGame
外星人游戏

#### 功能结构
##### 1.系统
 - 1.响应用户输入功能（响应按键）
 - 2.设置快捷键(退出-->Q、开始-->P)
 - 3.开始游戏功能
 - 4.结束游戏功能
 - 5.隐藏和显示光标
 - 6.移除子弹功能
 - 7.限制子弹数量
 - 8.修改属性设置
 
    
##### 2.设置  
- 1.屏幕设置
- 2.飞船设置
- 3.子弹设置
- 4.外星人设置
- 5.加快游戏节奏


##### 3.飞船
- 1.移动功能
- 2.发射子弹功能


##### 4.外星人
- 1.移动功能

##### 5.开始按钮
- 1.重置游戏功能
- 2.开始游戏功能


##### 6.积分
- 1.显示得分
- 2.显示最高分功能
- 3.等级功能
- 4.显示余下的飞船数
